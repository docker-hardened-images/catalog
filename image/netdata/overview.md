## About Netdata

Netdata is a real-time performance monitoring, built to be installed on all your systems and containers. It provides
unparalleled insights into your systems' performance and health, with zero configuration required. Netdata is designed
to be installed everywhere and is fast, efficient, and extensible. It collects thousands of metrics from applications,
services, and system resources in real-time and presents them in beautiful, interactive visualizations.

Netdata features include:

- **Real-time monitoring**: Collects thousands of metrics per second with per-second granularity
- **Zero configuration**: Works out of the box with no setup required
- **Distributed architecture**: Scales from a single node to thousands of nodes
- **Machine learning**: Detects anomalies and sends intelligent alerts
- **Low resource usage**: Minimal CPU and memory footprint
- **Extensible**: Supports custom collectors and plugins

For more information, visit https://www.netdata.cloud/

## About Docker Hardened Images

Docker Hardened Images are built to meet the highest security and compliance standards. They provide a trusted
foundation for containerized workloads by incorporating security best practices from the start.

### Why use Docker Hardened Images?

These images are published with zero-known CVEs, include signed provenance, and come with a complete Software Bill of
Materials (SBOM) and VEX metadata. They're designed to secure your software supply chain while fitting seamlessly into
existing Docker workflows.
