## About Placement

Placement is a Kubernetes controller from the Open Cluster Management project that makes placement decisions for
workloads across managed clusters. It evaluates Placement APIs to determine which managed clusters should run specific
workloads based on cluster labels, taints, tolerations, and scheduling policies.

The Placement controller supports features like prioritized scheduling, spread placement across failure domains, and
add-on-based scoring for intelligent workload distribution across multi-cluster environments.

For more details, visit https://github.com/open-cluster-management-io/ocm.
