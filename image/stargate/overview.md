## About Stargate

Stargate is an open-source data API gateway that exposes Cassandra and DataStax Enterprise databases via REST, GraphQL,
gRPC, and Document APIs. It provides a unified interface for accessing database capabilities, enabling developers to
build modern applications without deep database expertise. Stargate simplifies database integration by offering multiple
API protocols and abstracting database complexity behind a single gateway.

With support for both development and production deployments, Stargate enables seamless database modernization and
cloud-native development patterns. It includes built-in authentication, health checks, and scalable clustering
capabilities for enterprise-grade deployments.

For more details, see https://stargate.io/.

## About Docker Hardened Images

Docker Hardened Images are built to meet the highest security and compliance standards. They provide a trusted
foundation for containerized workloads by incorporating security best practices from the start.

### Why use Docker Hardened Images?

These images are published with zero-known CVEs, include signed provenance, and come with a complete Software Bill of
Materials (SBOM) and VEX metadata. They're designed to secure your software supply chain while fitting seamlessly into
existing Docker workflows.
