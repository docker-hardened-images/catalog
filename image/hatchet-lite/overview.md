## About Hatchet Lite

Hatchet is an orchestration engine for background tasks, AI agents, and durable workflows, built on PostgreSQL. It
provides a durable task queue with observability, DAG-based workflow orchestration, rate limiting and concurrency
control, cron and one-time scheduling, event-based triggers, and a real-time web UI with logs and metrics.

Hatchet Lite bundles the engine, API server, and web UI into a single container — designed for development environments
and smaller production deployments where running the full multi-service Hatchet stack would be overkill.

## About Docker Hardened Images

Docker Hardened Images are built to meet the highest security and compliance standards. They provide a trusted
foundation for containerized workloads by incorporating security best practices from the start.

### Why use Docker Hardened Images?

These images are published with near-zero known CVEs, include signed provenance, and come with a complete Software Bill
of Materials (SBOM) and VEX metadata. They're designed to secure your software supply chain while fitting seamlessly
into existing Docker workflows.
