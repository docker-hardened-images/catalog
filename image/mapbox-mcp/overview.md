## About Mapbox MCP Server

The Mapbox MCP Server is a Model Context Protocol implementation that gives AI applications access to Mapbox's
geospatial APIs. It exposes tools for geocoding and reverse geocoding, points-of-interest search, multi-modal
directions, travel-time matrices, route optimization, map matching, isochrone generation, static map images, and a range
of offline geospatial calculations.

For more details, visit https://github.com/mapbox/mcp-server.
