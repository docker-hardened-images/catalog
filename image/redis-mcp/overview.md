## About Redis MCP Server

Redis MCP Server is a Model Context Protocol server for Redis. It exposes tools that let MCP clients and AI agents
manage, query, and search Redis data through Redis connection settings supplied at runtime.

This Docker Hardened Image packages the upstream `redis-mcp-server` Python application with DHI Python runtimes for
Debian 13 and Alpine 3.24.
