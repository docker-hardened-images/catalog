## About ClickHouse Operator

The ClickHouse Operator by Altinity is an open-source Kubernetes operator that automates the deployment, scaling,
upgrade, and management of ClickHouse clusters on Kubernetes. It simplifies running ClickHouse in production by
providing declarative configuration, automated resource management, and seamless integration with Kubernetes-native
workflows. The operator enables organizations to run highly available, scalable, and manageable ClickHouse clusters with
minimal manual intervention.

For more information, visit the official repository: https://github.com/Altinity/clickhouse-operator

## About Docker Hardened Images

Docker Hardened Images are built to meet the highest security and compliance standards. They provide a trusted
foundation for containerized workloads by incorporating security best practices from the start.

### Why use Docker Hardened Images?

These images are published with near-zero known CVEs, include signed provenance, and come with a complete Software Bill
of Materials (SBOM) and VEX metadata. They're designed to secure your software supply chain while fitting seamlessly
into existing Docker workflows.
