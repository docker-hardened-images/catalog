## About DataHub GMS

DataHub GMS (General Metadata Service) is the core backend service of the DataHub open-source AI data catalog. It
provides a unified REST and GraphQL API for ingesting, storing, and querying metadata across datasets, dashboards,
pipelines, ML models, and other data assets throughout an organization's data ecosystem. DataHub GMS is a Spring Boot
application that integrates with Apache Kafka for event-driven metadata propagation, Elasticsearch or OpenSearch for
full-text search and indexing, and optionally Neo4j for graph-based lineage and relationship traversal.

For more information about DataHub, visit https://datahub.com.
