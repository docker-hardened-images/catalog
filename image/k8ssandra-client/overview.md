## About K8ssandra Client

K8ssandra Client is a kubectl plugin for managing K8ssandra clusters. K8ssandra is a cloud-native distribution of Apache
Cassandra that runs on Kubernetes. The plugin provides commands for cluster lifecycle management, nodetool operations,
data plane registration, and capacity estimation.

Features include:

- Manage K8ssandra clusters via kubectl
- Execute nodetool commands on Cassandra pods
- Register data planes into the control plane
- Cluster lifecycle operations (start, stop, restart)
- Capacity estimation for datacenter expansion

This image also includes kubectl for general Kubernetes operations.

For more information, visit the [k8ssandra-client GitHub repository](https://github.com/k8ssandra/k8ssandra-client).
