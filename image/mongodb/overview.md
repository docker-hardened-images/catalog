# About MongoDB

MongoDB is a popular open-source NoSQL database designed for scalability, high performance, and ease of development. It
stores data in flexible, JSON-like documents and is widely used for modern applications requiring fast, flexible data
access.

For more details, visit https://www.mongodb.com/docs/.

## About Docker Hardened Images

Docker Hardened Images are built to meet the highest security and compliance standards. They provide a trusted

These images are published with zero-known CVEs, include signed provenance, and come with a complete Software Bill of
