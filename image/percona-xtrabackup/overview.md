## About Percona XtraBackup

Percona XtraBackup is an open source backup and recovery tool for MySQL-compatible databases that enables fast,
non-blocking physical backups while databases remain online. It works by copying database files directly and applying
recorded changes to produce consistent, restorable backups, making it well suited for production environments where
downtime must be minimized.

Additional information about Percona software packages and repositories is available at:
https://docs.percona.com/percona-software-repositories/

## About Docker Hardened Images

Docker Hardened Images are built to meet the highest security and compliance standards. They provide a trusted

These images are published with zero-known CVEs, include signed provenance, and come with a complete Software Bill of
