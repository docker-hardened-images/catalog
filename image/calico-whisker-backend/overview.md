## About Calico Whisker Backend

Whisker Backend is the API service for the Calico Whisker network flow visualization UI. It exposes HTTP endpoints that
proxy and transform flow telemetry from Goldmane for display in the Whisker frontend. The Tigera Operator deploys this
image alongside the Whisker UI when observability features are enabled.

For complete documentation, architecture details, and deployment guides, see the official Calico documentation at
https://docs.tigera.io/calico.
