## About K8ssandra Cass Operator

The K8ssandra Cass Operator is a Kubernetes operator that automates the deployment and management of Apache Cassandra
clusters in Kubernetes environments. It handles provisioning of datacenters, scaling operations, rolling restarts,
upgrades, and container failure remediation, making it easier to run production-grade Cassandra workloads on Kubernetes.
The operator is part of the K8ssandra project and can be used standalone or as a component within the broader
k8ssandra-operator for multi-cluster deployments.

For more information, visit the [K8ssandra documentation](https://docs.k8ssandra.io/components/cass-operator/).
