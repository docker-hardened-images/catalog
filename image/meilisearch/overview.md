## About Meilisearch

Meilisearch is an open-source, RESTful search API that delivers lightning-fast, typo-tolerant, full-text and hybrid
search. It is designed to be embedded into applications with minimal configuration: indexes are schemaless, defaults are
production-sensible, and queries return relevant results in milliseconds. Meilisearch supports faceted filtering,
geosearch, multi-tenancy, and AI-powered hybrid search that combines semantic vector search with traditional full-text
matching. The server exposes an HTTP API on port 7700 and ships alongside `meilitool`, a utility for offline maintenance
of the Meilisearch data directory.
