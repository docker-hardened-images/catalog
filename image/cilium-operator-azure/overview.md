## About Cilium Operator Azure

The Cilium Operator Azure is a specialized component within the Cilium ecosystem designed for Azure Kubernetes Service
(AKS) and Azure-based Kubernetes clusters. It manages cluster-wide networking duties with deep Azure integration,
providing centralized functionality including Azure IPAM for IP address management within Azure VNets, coordination with
Azure Load Balancer and Virtual Machine Scale Sets, identity management, service load balancing, and network policy
enforcement. The operator acts as a control plane component that coordinates Cilium agents across nodes while leveraging
Azure-native networking features, ensuring consistent policy application and resource management throughout the cluster.

## About Docker Hardened Images

Docker Hardened Images are built to meet the highest security and compliance standards. They provide a trusted
foundation for containerized workloads by incorporating security best practices from the start.

### Why use Docker Hardened Images?

These images are published with near-zero known CVEs, include signed provenance, and come with a complete Software Bill
of Materials (SBOM) and VEX metadata. They're designed to secure your software supply chain while fitting seamlessly
into existing Docker workflows.
