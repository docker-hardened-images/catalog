## About socat

socat (SOcket CAT) is a multipurpose relay tool for bidirectional data transfer between two independent data channels.
It supports a wide variety of address types including TCP, UDP, Unix sockets, TLS/SSL, pipes, file descriptors, and
program execution. socat is commonly used for port forwarding, protocol translation, network debugging, and connecting
disparate network endpoints.

For more information, visit http://www.dest-unreach.org/socat/.
