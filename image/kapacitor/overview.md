## About Kapacitor

Kapacitor is a real-time streaming data processing engine for time-series data with native alerting and anomaly
detection capabilities. It ingests metrics from InfluxDB and other sources, processes them using TICKscript or
user-defined functions, and triggers alerts or actions based on configurable thresholds and patterns. It provides
built-in integrations with common alerting platforms and supports complex data transformations, downsampling, and custom
processing pipelines for monitoring and observability workflows.

For more details, see https://www.influxdata.com/.
