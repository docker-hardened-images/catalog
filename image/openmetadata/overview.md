## About OpenMetadata

OpenMetadata is an open source platform for data discovery, observability, and governance. It provides a unified
metadata store, APIs, and a web UI for cataloging data assets, tracking column-level lineage, running data quality
checks, and collaborating across teams.

This image packages the OpenMetadata server (a Java/Dropwizard service that also serves the web UI). It requires an
external metadata database (MySQL or PostgreSQL) and a search backend (Elasticsearch or OpenSearch).
