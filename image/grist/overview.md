## About Grist

Grist is a modern, open-source platform that blends the simplicity of spreadsheets with the power of databases. It
enables users to create custom applications tailored to their workflows, offering features like relational data
management, flexible layouts, and advanced formula support. With Grist's AI Assistant, users can interact with their
data using natural language, simplifying tasks such as formula generation and data analysis. The platform also provides
granular access controls, ensuring data security and collaboration efficiency. Whether for small businesses, educational
institutions, or research labs, Grist offers a scalable and customizable solution for data management needs.

For more details, visit https://getgrist.com/.

## About Docker Hardened Images

Docker Hardened Images are built to meet the highest security and compliance standards. They provide a trusted
foundation for containerized workloads by incorporating security best practices from the start.

### Why use Docker Hardened Images?

These images are published with near-zero known CVEs, include signed provenance, and come with a complete Software Bill
of Materials (SBOM) and VEX metadata. They're designed to secure your software supply chain while fitting seamlessly
into existing Docker workflows.
