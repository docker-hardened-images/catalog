## About Piraeus Server

Piraeus Server is the LINSTOR server component that combines the controller, satellite, and client functionality for
Kubernetes storage orchestration. It provides distributed, replicated block storage using DRBD technology, enabling
highly available persistent volumes for stateful applications. The server manages storage resources, handles
replication, and coordinates with other LINSTOR components to deliver enterprise-grade storage capabilities.
