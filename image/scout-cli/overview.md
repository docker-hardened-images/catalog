## About Docker Scout CLI

Docker Scout CLI is a powerful command-line tool for analyzing container images and identifying security
vulnerabilities. It provides comprehensive image analysis, policy evaluation, and supply chain security features to help
you build and maintain secure containerized applications.

The CLI offers commands for CVE detection, SBOM generation, policy compliance checking, and integration with CI/CD
pipelines. It supports analyzing local images, remote registries, and can compare images to identify security
improvements or regressions between versions.

For more details, visit https://github.com/docker/scout-cli.

## About Docker Hardened Images

Docker Hardened Images are built to meet the highest security and compliance standards. They provide a trusted
foundation for containerized workloads by incorporating security best practices from the start.

### Why use Docker Hardened Images?

These images are published with zero-known CVEs, include signed provenance, and come with a complete Software Bill of
Materials (SBOM) and VEX metadata. They're designed to secure your software supply chain while fitting seamlessly into
existing Docker workflows.
