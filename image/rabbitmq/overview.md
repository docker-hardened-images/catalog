## About RabbitMQ

RabbitMQ is open source message broker software (sometimes called message-oriented middleware) that implements the
Advanced Message Queuing Protocol (AMQP). The RabbitMQ server is written in the Erlang programming language and is built
on the Open Telecom Platform framework for clustering and failover. Client libraries to interface with the broker are
available for all major programming languages.

For more details, visit https://www.rabbitmq.com/.
