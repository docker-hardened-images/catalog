## About Telegraf

Telegraf is a plugin-driven server agent for collecting, processing, aggregating, and writing metrics. Built by
InfluxData, it supports over 300 input plugins for collecting metrics from systems, services, and third-party APIs, and
can output to a wide range of datastores including InfluxDB, Prometheus, and more.

This Docker Hardened Image provides a secure, minimal environment for running Telegraf as a metrics collection agent in
containerized workflows and Kubernetes deployments.
