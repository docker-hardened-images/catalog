## About Perplexity MCP Server

The Perplexity MCP Server is the official Model Context Protocol server for the Perplexity API Platform, bridging AI
assistants such as Claude, Cursor, and VS Code Copilot to Perplexity's real-time web search and reasoning services. It
exposes four MCP tools — `perplexity_search` (web search), `perplexity_ask` (conversational chat via sonar-pro),
`perplexity_research` (deep research with streaming via sonar-deep-research), and `perplexity_reason` (structured
reasoning via sonar-reasoning-pro) — over an HTTP/StreamableHTTP transport by default.

For more details, visit https://docs.perplexity.ai/guides/mcp-server.

## About Docker Hardened Images

Docker Hardened Images are built to meet the highest security and compliance standards. They provide a trusted
foundation for containerized workloads by incorporating security best practices from the start.

### Why use Docker Hardened Images?

These images are published with near-zero known CVEs, include signed provenance, and come with a complete Software Bill
of Materials (SBOM) and VEX metadata. They're designed to secure your software supply chain while fitting seamlessly
into existing Docker workflows.
