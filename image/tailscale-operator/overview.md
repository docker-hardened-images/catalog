## About the Tailscale Kubernetes Operator

The Tailscale Kubernetes Operator manages Tailscale resources from within a Kubernetes cluster. It can expose Kubernetes
Services on your tailnet, run as a subnet router or exit node, generate Tailscale-authenticated proxies for ingress and
egress, and let cluster workloads consume Tailscale identities through standard Kubernetes APIs.

The operator is part of the Tailscale platform, which is a mesh VPN built on the open source WireGuard protocol that
lets devices and services connect directly to each other without exposing them to the public internet.

For more details, visit
[https://tailscale.com/kb/1236/kubernetes-operator](https://tailscale.com/kb/1236/kubernetes-operator).

## About Docker Hardened Images

Docker Hardened Images are built to meet the highest security and compliance standards. They provide a trusted
foundation for containerized workloads by incorporating security best practices from the start.

### Why use Docker Hardened Images?

These images are published with zero-known CVEs, include signed provenance, and come with a complete Software Bill of
Materials (SBOM) and VEX metadata. They're designed to secure your software supply chain while fitting seamlessly into
existing Docker workflows.
