## About K8ssandra Operator

K8ssandra Operator is a Kubernetes operator that manages the lifecycle of Apache Cassandra clusters deployed on
Kubernetes. It provides multi-cluster and multi-region support, enabling distributed Cassandra deployments across
multiple Kubernetes clusters. The operator automates deployment, scaling, backup/restore operations, and integrates
monitoring and metrics collection. It introduces the K8ssandraCluster custom resource for declarative cluster management
and supports integration with Stargate (API gateway), Reaper (repair service), and Medusa (backup/restore). Official
documentation: https://docs.k8ssandra.io/
