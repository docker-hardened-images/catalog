## About node-collector

node-collector is a Kubernetes node information collector from Aqua Security's
[k8s-node-collector](https://github.com/aquasecurity/k8s-node-collector) project. It runs as a short-lived Kubernetes
Job to gather file system and process data from cluster nodes for CIS benchmark compliance checking.
