## About Gitleaks

Gitleaks is a SAST (Static Application Security Testing) tool for detecting and preventing hardcoded secrets like
passwords, API keys, and tokens in git repositories. Written in Go, Gitleaks is fast, lightweight, and portable, making
it easy to integrate into CI/CD pipelines and development workflows.

With over 20 million Docker downloads and thousands of GitHub stars, Gitleaks is a trusted solution among security
professionals, enterprises, and developers for protecting sensitive information from being committed to version control.

For more details, see https://github.com/gitleaks/gitleaks.

## About Docker Hardened Images

Docker Hardened Images are built to meet the highest security and compliance standards. They provide a trusted
foundation for containerized workloads by incorporating security best practices from the start.

### Why use Docker Hardened Images?

These images are published with zero-known CVEs, include signed provenance, and come with a complete Software Bill of
Materials (SBOM) and VEX metadata. They're designed to secure your software supply chain while fitting seamlessly into
existing Docker workflows.
