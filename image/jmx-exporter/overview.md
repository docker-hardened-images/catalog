## About JMX Exporter

JMX Exporter is a collector that captures Java Management Extensions (JMX) MBean values and exposes them in a
Prometheus-compatible format. It can run as a Java Agent inside your application or as a standalone service connecting
over RMI.

For more details, visit https://prometheus.github.io/jmx_exporter/.

## About Docker Hardened Images

Docker Hardened Images are built to meet the highest security and compliance standards. They provide a trusted

These images are published with near-zero known CVEs, include signed provenance, and come with a complete Software Bill
