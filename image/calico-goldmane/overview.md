## About Calico Goldmane

Goldmane is Calico's flow aggregation service. It collects, aggregates, and serves network flow telemetry for Calico's
observability features. Goldmane runs as a Kubernetes Deployment in the `calico-system` namespace and is managed by the
Tigera Operator when flow logging is enabled.

For complete documentation, architecture details, and deployment guides, see the official Calico documentation at
https://docs.tigera.io/calico.
