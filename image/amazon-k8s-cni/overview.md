## About amazon-vpc-cni-k8s

Networking plugin for pod networking in Kubernetes using Elastic Network Interfaces on AWS.

This plugin gives every pod a real VPC IP address by managing ENI attachment and IP allocation on each node, enabling
pods to communicate directly within the VPC without any overlay network.
